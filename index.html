About Me

Welcome! I am a software engineer and avid disc golfer. I have been writing automation software and supporting web applications for years, however I have yet to take a deep dive into full-stack web development... until now.

This is my third webpage, and it was built from scratch using HTML and CSS (without the help of any fancy frameworks). This page is fully responsive; Feel free to check out its various presentations across all devices (Desktop / Laptop / Tablet / Phone). As I continue to hone my skills, I am sure this site will progressively become more awesome. Just wait and see!
